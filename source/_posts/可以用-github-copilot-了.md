笔者在几乎一个月前申请了 GitHub Copilot 这个好用的服务，现在已经可以用了，这个服务的好处是可以自动帮你补全很多代码，你只要为你的函数起一个名字，就可以自动补全了。

比如下面的这段 Python 代码的函数主体部分就是 Copilot 补全的。

```python
def calcDaysBetwwenDates(startDate, endDate):
    return (endDate - startDate).days
```
